<Card className="mb-6">
  <CardHeader>
    <CardTitle>문자열 리터럴</CardTitle>

    </CardDescription>
  </CardHeader>
  <CardContent>
    ```js
    // 작은따옴표 사용
    const text1 = 'Hello'
    // 큰따옴표 사용
    const text2 = "World"
    ```

    ## 특수 문자 처리
    
    ```js
    // 16진수 이스케이프
    console.log("\xA9") // "©" 출력
    
    // 유니코드 이스케이프
    console.log("\u00A9") // "©" 출력
    
    // 유니코드 코드 포인트
    console.log("\u{2F804}") // 더 큰 범위의 유니코드 문자 표현 가능
    ```
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>템플릿 리터럴</CardTitle>
    <CardDescription>
      백틱(`)을 사용한 더 강력한 문자열 표현 방식
    </CardDescription>
  </CardHeader>
  <CardContent>
    ## 여러 줄 문자열
    ```js
    // 기존 방식
    const oldWay = "첫 번째 줄\n" +
                   "두 번째 줄";

    // 템플릿 리터럴 사용
    const newWay = `첫 번째 줄
    두 번째 줄`;
    ```

    ## 표현식 삽입
    ```js
    const name = "홍길동";
    const age = 20;

    // 기존 방식
    console.log("제 이름은 " + name + "이고, " + age + "살입니다.");

    // 템플릿 리터럴 사용
    console.log(`제 이름은 ${name}이고, ${age}살입니다.`);
    ```
  </CardContent>
</Card>

<Card className="mb-6">

          </li>

            <code>substring()</code>: 부분 문자열 추출
          </li>
          <li>
            <code>slice()</code>: 문자열의 일부분 추출
          </li>
        </ul>
      </div>
      <div>
        <h3 className="text-lg font-semibold mb-2">검색과 변환</h3>
        <ul className="list-disc list-inside space-y-2">

            <code>indexOf()</code>: 문자열 검색
          </li>
          <li>
            <code>replace()</code>: 문자열 치환
          </li>
          <li>
            <code>toLowerCase()/toUpperCase()</code>: 대소문자 변환
          </li>
        </ul>
      </div>

    ```js
    const text = "Hello, World!";
    
    console.log(text.charAt(0));        // "H"
    console.log(text.substring(0, 5));  // "Hello"
    console.log(text.indexOf("World")); // 7
    ```
