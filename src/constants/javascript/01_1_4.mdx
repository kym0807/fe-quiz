<Card className="mb-6">
  <CardHeader>
    <CardTitle>할당 연산자</CardTitle>
    <CardDescription>
      변수에 값을 할당하는 다양한 방법을 알아봅시다.
    </CardDescription>
  </CardHeader>
  <CardContent>
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
      <div className=" p-4 rounded-lg">
        ### 기본 할당
        ```js
        x = y    // 기본 할당
        x += y   // x = x + y
        x -= y   // x = x - y
        x *= y   // x = x * y
        x /= y   // x = x / y
        ```
      </div>
      
      <div className=" p-4 rounded-lg">
        ### 고급 할당
        ```js
        x ??= y  // x가 null/undefined면 y 할당
        x &&= y  // x가 truthy면 y 할당
        x ||= y  // x가 falsy면 y 할당
        ```
      </div>
    </div>

    ### 객체 속성 할당
    ```js
    const obj = {};
    
    obj.x = 3;        // 점 표기법
    const key = "y";
    obj[key] = 5;     // 대괄호 표기법
    
    console.log(obj); // { x: 3, y: 5 }
    ```
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>비교 연산자</CardTitle>
    <CardDescription>
      두 값을 비교하고 논리값을 반환하는 연산자들을 알아봅시다.
    </CardDescription>
  </CardHeader>
  <CardContent>
    <div className="grid grid-cols-1 lg:grid-cols-2 gap-4">
      <div className=" p-4 rounded-lg">
        ### 동등 비교
        ```js
        x == y    // 값이 같음
        x === y   // 값과 타입이 같음
        x != y    // 값이 다름
        x !== y   // 값 또는 타입이 다름
        ```
      </div>
      
      <div className=" p-4 rounded-lg">
        ### 크기 비교
        ```js
        x > y     // 보다 큼
        x >= y    // 크거나 같음
        x < y     // 보다 작음
        x <= y    // 작거나 같음
        ```
      </div>
    </div>

    <div className=" p-4 rounded-lg mt-4">
      ### 주의사항
      - == 보다는 === 사용을 권장
      - === 는 타입 변환을 하지 않음
      - 문자열은 유니코드 값으로 비교
    </div>
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>산술 연산자</CardTitle>
    <CardDescription>
      수학적 계산을 수행하는 연산자들을 알아봅시다.

  </CardHeader>
  <CardContent>
    ### 기본 연산자
    ```js
    x + y    // 덧셈
    x - y    // 뺄셈
    x * y    // 곱셈
    x / y    // 나눗셈
    x % y    // 나머지
    x ** y   // 거듭제곱
    ```

    ### 증감 연산자
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4 mt-4">
      <div className=" p-4 rounded-lg">
        ```js
        let x = 3;
        ++x;    // x를 증가시키고 반환
        x++;    // x를 반환하고 증가
        ```
      </div>
      <div className=" p-4 rounded-lg">
        ```js
        let y = 3;
        --y;    // y를 감소시키고 반환
        y--;    // y를 반환하고 감소
        ```
      </div>
    </div>

    ### 단항 연산자
    ```js
    +x      // 숫자로 변환
    -x      // 부호 반전
    ```

    <div className=" p-4 rounded-lg mt-4">
      ### 특별한 케이스
      - 0으로 나누면 Infinity 반환
      - 문자열 + 숫자는 문자열 연결
      - undefined와의 연산은 NaN
    </div>
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>논리 연산자</CardTitle>
    <CardDescription>
      불리언 값을 다루는 연산자와 단축 평가에 대해 알아봅시다.
    </CardDescription>
  </CardHeader>
  <CardContent>
    ### 기본 연산자
    ```js
    x && y   // 논리 AND
    x || y   // 논리 OR
    !x       // 논리 NOT
    x ?? y   // Nullish 병합
    ```

    ### 동작 방식
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4 mt-4">
      <div className=" p-4 rounded-lg">
        ```js
        // AND (&&)
        true && true    // true
        true && false   // false
        false && true   // false
        "강아지" && "고양이" // "고양이"
        ```
      </div>
      <div className=" p-4 rounded-lg">
        ```js
        // OR (||)
        true || false   // true
        false || true   // true
        "강아지" || "고양이" // "강아지"
        ```
      </div>
    </div>

    ### Nullish 병합 연산자
    ```js
    null ?? "기본값"      // "기본값"
    undefined ?? "기본값" // "기본값"
    false ?? "기본값"     // false
    0 ?? "기본값"        // 0
    ```

    <div className=" p-4 rounded-lg mt-4">
      ### 단축 평가
      - `falsy && anything`은 falsy 값 반환
      - `truthy || anything`은 truthy 값 반환
      - `nonNullish ?? anything`은 nonNullish 값 반환
    </div>
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>비트 연산자</CardTitle>
    <CardDescription>
      숫자의 이진 표현을 다루는 연산자들을 알아봅시다.
    </CardDescription>
  </CardHeader>
  <CardContent>
    ### 기본 연산자
    ```js
    a & b    // 비트 AND
    a | b    // 비트 OR
    a ^ b    // 비트 XOR
    ~a       // 비트 NOT
    a << b   // 왼쪽 시프트
    a >> b   // 오른쪽 시프트
    a >>> b  // 부호없는 오른쪽 시프트
    ```

    ### 실제 사용 예시
    ```js
    // 비트 AND 예시
    15 & 9   // 1111 & 1001 = 1001 (9)

    // 비트 시프트 예시
    9 << 2   // 1001 << 2 = 100100 (36)
    ```

    <div className=" p-4 rounded-lg mt-4">
      ### 활용 사례
      - 플래그 처리
      - 권한 관리
      - 성능 최적화
      - 2의 거듭제곱 계산
    </div>

    <div className=" p-4 rounded-lg mt-4">
      ### 주의사항
      - 32비트 정수로 처리됨
      - 음수 처리 시 주의 필요
      - 가독성이 떨어질 수 있음
    </div>
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>

    <CardDescription>

  </CardHeader>
  <CardContent>
    ### 문자열 연결
    ```js
    // 기본 연결
    ```

    ### 조건(삼항) 연산자
    ```js
    조건 ? 참일때값 : 거짓일때값

    // 예시
    const 상태 = 나이 >= 18 ? "성인" : "미성년자";
    const 메시지 = 로그인 ? "환영합니다" : "로그인해주세요";
    ```

      ### 활용 팁
      - 간단한 조건부 할당에 유용
      - 중첩 사용은 피하기
      - 가독성을 고려하여 사용
    </div>
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>특수 연산자들</CardTitle>
    <CardDescription>
      delete, typeof, instanceof 등 특별한 목적을 가진 연산자들을 알아봅시다.
    </CardDescription>
  </CardHeader>
  <CardContent>
    ### typeof 연산자
    ```js
    typeof 42           // "number"
    typeof "문자열"     // "string"
    typeof true        // "boolean"
    typeof undefined   // "undefined"
    typeof null        // "object" (JavaScript의 유명한 버그)
    typeof {}         // "object"
    typeof []         // "object"
    typeof function(){} // "function"
    ```

    ### instanceof 연산자
    ```js
    const 배열 = [];
    const 날짜 = new Date();

    배열 instanceof Array     // true
    날짜 instanceof Date      // true
    ```

    ### delete 연산자
    ```js
    const 사용자 = {
      이름: "김철수",
      나이: 25
    };

    delete 사용자.나이;     // true
    console.log(사용자);    // { 이름: "김철수" }
    ```

    <div className=" p-4 rounded-lg mt-4">
      ### 옵셔널 체이닝 (?.)
      ```js
      const 사용자 = { 
        주소: { 
          도시: "서울" 
        } 
      };

      console.log(사용자?.주소?.도시);  // "서울"
      console.log(사용자?.계좌?.번호);  // undefined
      ```
    </div>
  </CardContent>
</Card>
