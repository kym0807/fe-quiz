## 클래스와 생성자  
<Card className="mb-6">  
  <CardHeader>  
    <CardTitle>클래스 선언</CardTitle>  
    <CardDescription>  
      `class` 키워드와 생성자를 활용하여 클래스를 선언합니다.  
    </CardDescription>  
  </CardHeader>  
  <CardContent>  
    ```javascript  
    class Person {  
      name;  

      constructor(name) {  
        this.name = name;  
      }  

      introduceSelf() {  
        console.log(`Hi! I'm ${this.name}`);  
      }  
    }  

    const giles = new Person("Giles");  
    giles.introduceSelf(); // Hi! I'm Giles  
    ```  
    - `constructor`는 클래스의 생성자로, 객체를 초기화합니다.  
    - `this`는 생성된 객체를 참조합니다.  
  </CardContent>  
</Card>  

<Card className="mb-6">  
  <CardHeader>  
    <CardTitle>생성자 생략</CardTitle>  
    <CardDescription>  
      특별한 초기화가 필요 없다면 생성자를 생략할 수 있습니다.  
    </CardDescription>  
  </CardHeader>  
  <CardContent>  
    ```javascript

        console.log("zzzzzzz");
    ```  

  </CardContent>  
</Card>  

## 상속  
<Card className="mb-6">  
  <CardHeader>  
    <CardTitle>클래스 상속</CardTitle>  
    <CardDescription>  
      `extends` 키워드를 사용하여 상속을 구현합니다.  
    </CardDescription>  
  </CardHeader>  
  <CardContent>  
    ```javascript  
    class Person {  
      constructor(name) {  
        this.name = name;  
      }  

      introduceSelf() {  
        console.log(`Hi! I'm ${this.name}`);  
      }  
    }  

    class Professor extends Person {  
      teaches;  

      constructor(name, teaches) {  
        super(name);  
        this.teaches = teaches;  
      }  

      introduceSelf() {  
        console.log(`My name is ${this.name}, and I will be your ${this.teaches} professor.`);  
      }  

      grade(paper) {  
        const grade = Math.floor(Math.random() * 5 + 1);  
        console.log(grade);  
      }  
    }  

    const walsh = new Professor("Walsh", "Psychology");  
    walsh.introduceSelf(); // My name is Walsh, and I will be your Psychology professor.  
    walsh.grade("paper"); // 랜덤 점수  
    ```  
    - `super()`는 부모 클래스의 생성자를 호출합니다.  
    - 자식 클래스에서 부모 클래스의 메서드를 재정의할 수 있습니다.  
  </CardContent>  
</Card>  

## 캡슐화  
<Card className="mb-6">  
  <CardHeader>  
    <CardTitle>캡슐화와 private 속성</CardTitle>  
    <CardDescription>  
      `#` 접두사를 사용해 속성과 메서드를 비공개로 설정합니다.  
    </CardDescription>  
  </CardHeader>  
  <CardContent>  
    ```javascript  
    class Student extends Person {  
      #year;  

      constructor(name, year) {  
        super(name);  
        this.#year = year;  
      }  

      introduceSelf() {  
        console.log(`Hi! I'm ${this.name}, and I'm in year ${this.#year}.`);  
      }  

      canStudyArchery() {  
        return this.#year > 1;  
      }  
    }  

    const summers = new Student("Summers", 2);  
    summers.introduceSelf(); // Hi! I'm Summers, and I'm in year 2.  
    summers.canStudyArchery(); // true  
    summers.#year; // SyntaxError  
    ```  
    - `#`는 private 속성을 선언합니다.  
    - private 속성은 클래스 외부에서 접근할 수 없습니다.  
  </CardContent>  
</Card>  

<Card className="mb-6">  
  <CardHeader>  
    <CardTitle>private 메서드</CardTitle>  
    <CardDescription>  
      메서드도 private로 선언할 수 있습니다.  
    </CardDescription>  
  </CardHeader>  
  <CardContent>  
    ```javascript  
    class Example {  
      somePublicMethod() {  
        this.#somePrivateMethod();  
      }  

      #somePrivateMethod() {  
        console.log("You called me?");  
      }  
    }  

    const example = new Example();  
    example.somePublicMethod(); // 'You called me?'  
    example.#somePrivateMethod(); // SyntaxError  
    ```  
    private 메서드는 객체 외부에서 호출할 수 없습니다.  
  </CardContent>  
</Card>
