<Card className="mb-6">
  <CardHeader>
    <CardTitle>배열 메서드</CardTitle>
    <CardDescription>
      JavaScript 배열에서 자주 사용되는 유용한 메서드들을 소개합니다
    </CardDescription>
  </CardHeader>
  <CardContent>
    <div className="grid grid-cols-1 md:grid-cols-2 gap-6">
      <div>
        <h3 className="text-lg font-semibold mb-3">배열 수정</h3>
        <div className="space-y-4">
          ```js
          // 끝에 추가 (push)
          const arr1 = ["a", "b"];
          arr1.push("c");    // ["a", "b", "c"]
          
          // 앞에 추가 (unshift)
          arr1.unshift("z"); // ["z", "a", "b", "c"]
          
          // 끝에서 제거 (pop)
          const last = arr1.pop();  // last: "c"
          
          // 앞에서 제거 (shift)
          const first = arr1.shift(); // first: "z"
          ```
        </div>
      </div>
      
      <div>
        <h3 className="text-lg font-semibold mb-3">배열 탐색</h3>
        <div className="space-y-4">
          ```js
          const arr = ["a", "b", "c", "b", "d"];
          
          // 앞에서부터 검색
          arr.indexOf("b");    // 1
          
          // 뒤에서부터 검색
          arr.lastIndexOf("b"); // 3
          
          // 조건으로 검색
          arr.findIndex(x => x > "c"); // 4 ("d"의 인덱스)
          ```
        </div>
      </div>
    </div>
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>반복 메서드</CardTitle>
    <CardDescription>
      배열의 요소들을 순회하면서 작업을 수행하는 메서드들입니다
    </CardDescription>
  </CardHeader>
  <CardContent>
    <Tabs defaultValue="basic" className="mb-6">
      <TabsList>
        <TabsTrigger value="basic">기본 반복</TabsTrigger>
        <TabsTrigger value="transform">변환</TabsTrigger>
        <TabsTrigger value="filter">필터링</TabsTrigger>
      </TabsList>
      
      <TabsContent value="basic">
        ```js
        const items = ["A", "B", "C"];

        // forEach: 각 요소에 대해 함수 실행
        items.forEach(item => console.log(item));
        
        // for...of: 최신 문법 (권장)
        for (const item of items) {
          console.log(item);
        }
        ```
      </TabsContent>
      
      <TabsContent value="transform">
        ```js
        const numbers = [1, 2, 3];

        // map: 각 요소를 변환한 새 배열 반환
        const doubled = numbers.map(x => x * 2);
        // [2, 4, 6]
        
        // flatMap: map 후 중첩 배열 펼치기
        const pairs = numbers.flatMap(x => [x, x]);
        // [1, 1, 2, 2, 3, 3]
        ```
      </TabsContent>
      
      <TabsContent value="filter">
        ```js
        const scores = [45, 90, 65, 85, 30];

        // filter: 조건에 맞는 요소만 선택
        const passing = scores.filter(x => x >= 60);
        // [90, 65, 85]
        
        // find: 조건에 맞는 첫 번째 요소
        const first = scores.find(x => x >= 60);
        // 90
        ```
      </TabsContent>
    </Tabs>
  </CardContent>
</Card>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>다차원 배열</CardTitle>
    <CardDescription>
      배열 안에 배열을 포함하는 구조입니다
    </CardDescription>
  </CardHeader>
  <CardContent>
    ```js
    // 2차원 배열 생성
    const matrix = [
      [1, 2, 3],
      [4, 5, 6],
      [7, 8, 9]
    ];

    // 접근하기
    console.log(matrix[0][1]); // 2
    console.log(matrix[1][1]); // 5

    // 2차원 배열 순회
    for (const row of matrix) {
      for (const item of row) {
        console.log(item);
      }
    }
    ```
  </CardContent>

        ```js
        // 1. Array 생성자 사용
        const arr1 = Array(3); // [empty × 3]

        const arr2 = [1, , 3]; // [1, empty, 3]
        ```

        ```js
        const arr = [1, , 3];

        // forEach는 빈 슬롯 건너뜀

        // for...of는 undefined로 처리

          console.log(x); // 1, undefined, 3
        }
        ```

    </Tabs>

<Card className="mb-6">
  <CardHeader>
    <CardTitle>배열과 유사 배열</CardTitle>
    <CardDescription>
      배열처럼 보이지만 실제 배열이 아닌 객체들입니다

  <CardContent>
    ```js
    // 유사 배열 객체
    const arrayLike = {
      0: "첫 번째",
      1: "두 번째",
      length: 2
    };

    // 배열 메서드 사용하기
    const realArray = Array.from(arrayLike);
    // 또는
    const realArray2 = Array.prototype.slice.call(arrayLike);

    // 문자열도 유사 배열처럼 동작
    Array.from("Hello"); // ["H", "e", "l", "l", "o"]
    ```
  </CardContent>
